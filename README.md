# Project IV: Lagrangian-Particle-Tracking
Matlab code for my 4th year project

The majority of code for this project can be found in SW_share. Code comparing reconstructed Lagrangian velocity fields to the actual Lagrangian velocity field can be found in Averaging. The scripts in Project IV are for plots used in my project report. 

All scripts from Cai are in clearly marked folders: ('SW_share/__MACOSX','SW_share/SW_share/Cai_SW', 'SW_share/SW_share/cmocean, 'SW_share/SW_share/rfftCai', 'Averaging/averaging/__MACOSX', 'Averaging/averaging/Cai_Averaging','Averaging/averaging/filters'). 

All other scripts are my own, with the exception of ICMultOptions.m, driver_SW.m and driver_SW_test.m which are adapted from code given to me by Cai.This is discussed in the comments on these files. 

Pictures from the project can be found in 'SW_share/SW_share/Pictures project', they are grouped (hopefully logically!). Some of the pictures included contain errors from previous itterations of code before it was corrected. 

Files are zipped as they are relatively large. To view a file from the repository click it and then click 'view raw' which will download the zipped file. 

The .mat files have not been included due to size limitations.
